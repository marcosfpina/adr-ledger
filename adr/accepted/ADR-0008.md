## Context

CEREBRO is a GenAI credit validation & knowledge extraction engine optimized for **consuming GCP Trial Credits** ($300-10k) efficiently.

### Problem: Credits Expiring Unused

**Scenario**: Received $10k GCP credits (90-day expiration)

**Challenge**:
- Credits expire if not used within 90 days
- No programmatic way to check remaining balance before ML run
- Manual tracking leads to errors: "Did we already spend $8k or $9k?"

**Real Incident**:
```
Day 85: $2k credits remaining
Engineer: Starts $3k ML experiment (7-day training)
Day 90: Credits expire
Day 92: ML run fails mid-training (credits depleted)
Result: $2k wasted on incomplete run, zero learning
```

### Requirements

1. **Pre-Flight Validation**: Check credits available *before* starting expensive run
2. **Real-Time Tracking**: Know current spend at any moment
3. **Integration**: Plug into NEUTRON ML pipelines
4. **ROI Focus**: Maximize learning per dollar spent

## Decision

Implement **CerebroCreditValidator** with BigQuery audit integration.

### Architecture

```python
# src/phantom/modules/credit_burner/audit.py
class CerebroCreditValidator:
    """Validates GCP credits before expensive operations."""

    def __init__(self, project_id: str):
        self.bq_client = bigquery.Client(project=project_id)

    def check_credits_available(
        self,
        required_credits: float,
        threshold: float = 10.0  # $10 minimum buffer
    ) -> bool:
        """
        Query BigQuery billing export to check remaining credits.

        Returns:
            True if (remaining_credits >= required_credits + threshold)
            False otherwise
        """
        query = """
        SELECT
          SUM(cost) as total_spent,
          SUM(IF(credit_type = 'PROMOTION', ABS(credit_amount), 0)) as total_credits
        FROM `{project}.billing_export.gcp_billing_export_v1`
        WHERE
          DATE(usage_start_time) >= '{start_date}'
          AND DATE(usage_end_time) <= '{end_date}'
        """

        df = self.bq_client.query(query).to_dataframe()
        total_spent = df['total_spent'].iloc[0] or 0
        total_credits = df['total_credits'].iloc[0] or 0
        remaining = total_credits - total_spent

        if remaining < (required_credits + threshold):
            log.error(
                f"Insufficient credits: {remaining:.2f} available, "
                f"{required_credits:.2f} required (+ ${threshold} buffer)"
            )
            return False

        log.info(f"Credits OK: ${remaining:.2f} available")
        return True
```

from phantom.modules.credit_burner.audit import CerebroCreditValidator

class CostTracker:
    def __init__(self):
        self.validator = CerebroCreditValidator(project_id="my-gcp-project")

    def validate_before_run(self, estimated_cost: float):
        """Pre-flight check before ML pipeline."""
        if not self.validator.check_credits_available(estimated_cost):
            raise InsufficientCreditsError(
                f"Cannot start run: ${estimated_cost:.2f} required, "
                "insufficient credits"
            )

# roi_optimizer.py
def optimize_roi(budget: float, experiments: List[Experiment]):
    """
    Maximize learning per dollar.
    
    Algorithm:
    1. Estimate cost per experiment
    2. Estimate learning value (subjective score)
    3. Sort by value/cost ratio
    4. Run experiments until budget exhausted
    """
    experiments.sort(key=lambda e: e.value / e.cost, reverse=True)
    
    spent = 0
    for exp in experiments:
        if spent + exp.cost <= budget:
            run_experiment(exp)
            spent += exp.cost

### Cost Tracking Comparison

| Method                  | Latency | Accuracy | Cost       | Integration |
|-------------------------|---------|----------|------------|-------------|
| **Manual (spreadsheet)**| Hours   | ±20%     | $0         | None        |
| **GCP Billing UI**      | 24h lag | 100%     | $0         | None        |

| **Cloud Billing API**   | Real-time| 100%    | $0         | ✅ API (limited) |

**Winner**: BigQuery audit (programmatic + accurate + fast enough)

**Strategy 2 (ROI-Optimized)**: Prioritize high-value/cost experiments
