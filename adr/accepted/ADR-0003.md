## Context

SPECTRE significa **S**entiment & **P**attern **E**xtraction for **C**ontextual **T**extual **R**esearch & **E**valuation. É o "forensics engine" da stack - analisa texto para extrair insights que humanos não conseguem ver em escala.

Casos de uso críticos:

1. **Agent Auditing**: Avaliar qualidade de outputs de LLMs
2. **FOMO Detection**: Identificar hype vs. substância em dados de mercado
3. **Pattern Mining**: Extrair recurring themes de corpora massivos
4. **Sentiment Scoring**: Quantificar tom/mood de documentos

### Módulos Core

1. **Sentiment Analyzer**
   - Multi-model ensemble (VADER + BERT + domain-specific)
   - Output: score [-1, 1] + confidence + breakdown

2. **Pattern Extractor**
   - Regex-based para padrões conhecidos
   - ML-based para padrões emergentes
   - Topic modeling para clustering semântico

3. **Entity Resolver**
   - NER para entidades técnicas
   - Coreference resolution para contexto

### Agent Auditing

```python
@spectre.audit
def evaluate_agent_output(output: str, context: str) -> AuditReport:
    return {
```

## Rationale

### Por que separar de CEREBRO?

| Aspecto | CEREBRO | SPECTRE |
|---------|---------|---------|
| **Função** | Retrieval | Analysis |
| **Input** | Query | Document |
| **Output** | Relevant chunks | Insights/scores |
| **Escala** | Query-time | Batch + streaming |

São complementares: CEREBRO retrieves, SPECTRE analyzes.

### Modelo de Incentivos (DAO Future)

SPECTRE pode servir como **oracle de qualidade** para um sistema de incentivos:

```
Agent produz output → SPECTRE avalia → Score determina reward
```

Isso permite:
- **Self-improving agents**: Feedback loop de qualidade
- **Fair compensation**: Reward proporcional à qualidade
- **Accountability**: Audit trail de performance

## Consequences

### Integration com ADR Ledger

SPECTRE analisa as próprias ADRs para:
- Detectar sentiment drift ao longo do tempo
- Identificar padrões em decisões rejeitadas
- Sugerir melhorias em ADRs propostas

```bash
adr sync → spectre_corpus.json → SPECTRE analysis → insights
```

### Métricas

- **Sentiment Accuracy**: >85% agreement com human labels
- **Pattern Precision**: >80% valid patterns
- **Latency**: <100ms para single document
- **Throughput**: >1000 docs/sec batch
