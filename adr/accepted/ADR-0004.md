## Context

PHANTOM é o "fantasma na máquina" - o sistema que processa dados silenciosamente, classifica, transforma, e alimenta outros componentes. Sem ele, CEREBRO não teria dados limpos para indexar, SPECTRE não teria corpus preparado para análise.

O nome vem da capacidade de operar "invisível" no background, fazendo o trabalho pesado que ninguém vê mas todos dependem.

### Intelligence Layer

O diferencial do PHANTOM é a **Intelligence Layer** que adiciona reasoning antes de executar:

```python
class PhantomRouter:
    """
    Analisa query/task e decide o melhor caminho de execução.
    Evita trabalho redundante via shared context.
    """
    
    def route(self, task: Task) -> ExecutionPlan:
        # 1. Analisa intent
        intent = self.analyze_intent(task)
        
        # 2. Checa cache de resultados similares
        if cached := self.check_cache(intent):
            return CachedPlan(cached)
        
        # 3. Decompõe em subtasks
        subtasks = self.decompose(task, intent)
        
        # 4. Otimiza ordem de execução
        plan = self.optimize_dag(subtasks)
        
        return plan
```

### Classificadores Core

1. **Document Classifier**
   - Input: Documento raw
   - Output: Tipo (ADR, RFC, Bug Report, etc.)
   - Model: Fine-tuned BERT

2. **Priority Classifier**
   - Input: Task/Issue
   - Output: Urgência (P0-P4)
   - Model: Gradient Boosting + Rules

3. **Sensitivity Classifier**
   - Input: Qualquer texto
   - Output: PII risk score + detected entities
   - Model: spaCy NER + custom rules

## Rationale

### Por que não Airflow direto?

Airflow é excellent para:
- Workflows bem definidos e estáveis
- Large-scale batch processing
- Teams com DevOps dedicado

Mas para a stack inteligente, precisamos de:
- **Flexibility**: Workflows que mudam frequentemente
- **Intelligence**: Decisões em runtime, não só DAG estático
- **Lightweight**: Roda em single node ou cluster
- **Integration**: Tight coupling com CEREBRO/SPECTRE

PHANTOM pode eventualmente usar Airflow como **executor backend**, mas a camada de inteligência é própria.
